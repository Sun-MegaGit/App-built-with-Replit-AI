# App-built-with-Replit-AI
Exam Simulator
https://rac-exam-prep--snlvdk.replit.app/

This app is fully static it's a React app with all 100 questions bundled directly into the code. There's no backend, no database, no user accounts, and no API calls being made.

That means-

Unlimited users anyone with the link can use it simultaneously
Unlimited sessions each user can take the exam as many times as they like
No login required just open and go
No data stored results only exist in the browser while the session is open; nothing is sent anywhere
The only thing that would ever limit it is Replit's hosting infrastructure itself, but for a static site like this, it handles very high traffic without issue.
